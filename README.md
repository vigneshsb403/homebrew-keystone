# Homebrew Keystone Tap

Personal Homebrew tap for [Keystone](https://github.com/vigneshsb403/Keystone), a macOS clipboard snippet manager with menu bar integration.

## Installation

```bash
brew install vigneshsb403/keystone/keystone
```

## About

Keystone is a productivity tool that helps you quickly capture, search, and paste frequently used text snippets from your clipboard. Access it via:

- **Menu bar icon** - Always available in your macOS menu bar
- **Global hotkey** - Quick-capture snippets from any application
- **Search** - Find snippets in seconds
- **Export/Import** - Back up and restore your snippets as JSON

## Privacy

All your snippets are stored locally on your machine. Keystone does not collect or send any data to external servers.
